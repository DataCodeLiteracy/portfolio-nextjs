# 포트폴리오 웹사이트 with Next.js 13

### 요약

단일 페이지, 스크롤 인터렉션

### 시작하기

```bash
$ git clone https://github.com/DataCodeLiteracy/portfolio-nextjs.git
```

```bash
$ yarn
```

```bash
$ yarn build
```

```bash
$ yarn start
```
